# Screenshots Directory

This folder contains screenshots of the portfolio website for the README.

## Required Screenshots

### 1. home.png
**What to capture:** Full homepage showing:
- 3D hero section with floating shapes
- Typing animation text
- Navigation bar
- Scroll down to show multiple sections

**How to capture:**
1. Open http://localhost:5175
2. Wait for all animations to load
3. Press F12 → Device Toolbar → Set to Desktop (1920x1080)
4. Click three dots → Capture screenshot → Full size

### 2. dashboard.png
**What to capture:** Admin dashboard showing:
- Stats cards (projects, messages, views, visitors)
- Sidebar navigation
- Premium dark theme UI
- Any interactive elements

**How to capture:**
1. Go to /admin route
2. Login with credentials
3. Capture full dashboard view

### 3. mobile.png
**What to capture:** Mobile responsive view showing:
- Mobile navigation menu
- Responsive hero section
- Touch-friendly interface
- Any mobile-specific features

**How to capture:**
1. Open DevTools (F12)
2. Toggle device toolbar (Ctrl+Shift+M)
3. Select iPhone 12 Pro or similar
4. Capture screenshot

## Alternative Tools

### Manual Screenshot Tools:
- **Windows:** Win + Shift + S (Snipping Tool)
- **Browser Extensions:** Nimbus Screenshot, Awesome Screenshot
- **Online:** web-capture.net

### Automated Tools:
- **Puppeteer:** For programmatic screenshots
- **Playwright:** Cross-browser automation
- **Screenshot Guru:** API-based screenshots
